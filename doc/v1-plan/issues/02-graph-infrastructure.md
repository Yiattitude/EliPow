## What to build
建立系统的底层知识网络结构。后端需部署 Neo4j 数据库，定义领域、标准、知识点等五级网络模型，并支持图谱局部查询接口。前端需集成 AntV G6（或 Cytoscape.js）实现该大图谱的可视化，必须支持语义缩放（按比例聚类/展开），避免浏览器渲染上千节点卡顿。用户可以点击节点，在右侧抽屉面板中预览与该节点相关的规范文档资源。

## Acceptance criteria
- [ ] 后端 Neo4j 包含测试用五级知识节点数据，并能通过接口提供带关系的 JSON 图数据。
- [ ] 前端能用 AntV G6 渲染出后端返回的全量知识图谱网络结构。
- [ ] 图谱支持滚动鼠标缩放，缩小时节点自动聚类，放大时展示微观关联。
- [ ] 单击任意节点，屏幕右侧滑出侧边栏，展示该节点关联的教学视频或 PDF 标题。
