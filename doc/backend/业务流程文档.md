# 业务流程文档

## 1. 注册与图谱初始化
1. 用户注册/登录 (MySQL)
2. 采集初始画像与目标岗位 (MySQL)
3. **发起计算机自适应诊断测评 (CAT)**：
   - **Hub 节点优先抽样**：总控智能体抽取目标岗位图谱中入度/出度最高的核心骨干节点（如“电路基础”）进行测试。
   - **二分法收敛**：答对核心节点，自动将其前置依赖标记为“大概率掌握（跳过）”并向后跃迁测试更难节点；答错则向下钻取测试基础前置节点。
   - **强制熔断止损**：设定最大测评时间（如 15 分钟）或最大题量（如 15 题），达到阈值立刻停止并生成初始权重，避免用户疲劳。
4. 初始测评数据投递至 Neo4j，生成初始“个人知识图谱权重”

## 2. 测评与智能体路径推荐
1. **总控调度智能体**：接收到用户路径生成请求。
2. **知识检索智能体**：从 Neo4j 提取该用户目标岗位的要求子图，及个人当前图谱子图。
3. **推理智能体 (认知约束打底)**：在生成阶段性路径时，强制引入“认知负荷约束”。连续安排 3 个高难度新知识点后，必定插入 1 个“复习节点/缓冲带”。同时校验所有推荐节点是否绑定了底层实操/视频资源，若无资源则标记为“待补充”并延后排期。
4. **弹性重规划机制 (Elastic Re-planning)**：当用户偏离计划（未完成或跳过）时，不立即进行全局重算。设定偏差阈值（如累计落后>20%），仅微调后续顺序；超过阈值才触发全局重规划，保护学习信心。
5. **旁路探索**：允许用户在主线路径外，以低成本探索非必修的相关知识，增强学习兴趣。

## 3. 渐进式 GraphRAG 与苏格拉底问答流程
1. **用户提问与意图识别**：用户提出疑问，系统直接提取关键词。
2. **渐进式联合检索 (Hybrid Search GraphRAG)**：V1 阶段不强求全量图谱化。先在 Neo4j 图谱中命中“相关核心节点/章节”，然后在同一个 Cypher 查询中，直接利用 Neo4j 5.x 的原生 Vector Index 对该章节关联的 DocumentChunk 节点进行向量相似度召回，实现图结构与语义相似度的完美融合。
3. **单智能体伪苏格拉底 (Pseudo-Socratic Prompting)**：系统拼接检索到的图谱切片，并注入极度严苛的防剧透 System Prompt（绝对禁止直接给出答案）。大模型基于图谱约束，直接流式生成苏格拉底式反问。
4. **前端响应与溯源**：流式返回带有 `<ref id="xxx">` 强标签的响应，前端渲染高亮原文。

## 4. 产教融合资源入库流程
1. 企业文档/运维案例导入。
2. **知识库更新智能体**：提取文档中的操作步骤与应用场景。
3. 将提取的新节点自动挂载到 Neo4j 中现有的“技术领域/知识点”网络下。
4. （可选）触发重新评估受影响的岗位图谱模型。

## 5. 产教缺口对齐与图谱联动流程（教研端）
1. **跨层自动映射**：导入高校教学大纲后，系统利用 Embedding 向量相似度算法，将大纲章节自动映射至行业图谱的知识点，生成初步对齐锚点，再交由教研专家人工确认 (Human-in-the-loop)。
2. **交集与差集分析**：系统提取“目标岗位图谱”，并与“高校大纲图谱”做逻辑集合运算。
3. **可视化缺口报告**：向学生展示“交集”（学校教的，用于复习）与“差集”（学校没教但企业要的，用于自学补充）；向教师端提交“产教缺口沙盘”报告。

## 6. 教研端双轨数据流与图谱演化 (工程化优化)
1. **微观实时互动统计 (Event-Driven)**：对于随堂测验等高时效场景，前端答题流水进入 Redis Pub/Sub 或 RabbitMQ，由后台更新 Redis 中的 Hash 计数器（如 `HINCRBY class_101_quiz_5 correct_count 1`），供教师端实现毫秒级微观热力图刷新。
2. **宏观全局热力图 (T+1 离线聚合)**：防范 Neo4j 的全局广度聚合导致熔断。全专业/全班的整体能力模型，在凌晨通过跑批任务（Cron Job）将 Neo4j 的 BKT 权重聚合拍平到 MySQL/Redis 中，教师端仅读取离线缓存。
3. **A/B 测验证明增益**：系统并行运行简单滑动窗口策略与 BKT 模型进行对比，防范数据稀疏带来的 BKT 过拟合。
4. **异步削峰写回**：前端产生的零散学习行为流水通过 MQ 异步削峰，后台 Worker 批量更新 Neo4j 个人节点连线的 `mastery_score`。
5. **WebSocket 实时推送**：权重更新完成后，通过 WebSocket 向前端推送状态，推理智能体随即进行路径自动纠偏。
