# 前端页面与功能规划文档

## 1. 设计风格与交互基调 (Design System)
结合项目的技术极客感与教育专业性，前端 UI 采用以下基调：
- **色彩模式**：深色模式 (Dark Mode) 为主，支持浅色模式等多种主题切换，辅以科技蓝、紫作为高亮强调色 (Vibrant Accents)。
- **视觉风格**：现代化的玻璃拟物化设计 (Glassmorphism)，卡片使用半透明磨砂质感，搭配发光效果突出关键信息。
- **排版字体**：使用现代无衬线字体（如 Inter 或 Roboto），数字和数据面板要求高度清晰。
- **动效反馈**：增加微动效，特别是节点解锁、技能树点亮等时刻，增强沉浸感与成就感。

## 2. 核心功能页面划分

### 2.1 登录与注册页
**功能目标**：完成账号体系打通与用户初始学情偏好数据的采集，需降低冷启动摩擦。
- **1.1 快捷登录**：支持账号密码登录，并**接入微信和 QQ 扫码**，降低使用门槛。
- **1.2 渐进式动态画像**：采用允许“先跳过”的渐进式设计。通过分步卡片交互，让用户选择：
  - 年级与未来规划（考研、考公、就业）
  - 目标岗位或目标院校
  - **当前最薄弱的专业课/知识点**（为 AI 提供绝佳的冷启动基准）

### 2.2 仪表盘页 (Dashboard)
**功能目标**：用户每次登录后的学习主阵地，直观展示学习进度、提供实时行业动态，营造强烈的学习成就感。
- **2.1 布局原则**：
  - 上方：横向滚动式学习路径轴
  - 左下：岗位技能树点亮面板
  - 右下：实时信息咨询栏
- **2.2 横向“地铁图”路径轴 (Timeline)**：置于页面上方，采用水平可拖拽滚动的“地铁图”设计，节约纵向空间。同步用户选定的目标路径，由 AI 生成动态的时间规划建议。
- **2.3 岗位技能树点亮面板 (Skill Tree)**：摒弃传统雷达图，采用类似游戏设定的**“技能树点亮”或“核心能力充能进度条”**。直观展示目标岗位所需的理论、实操、安全等多维度的完成情况，用极其可视化的方式让用户获得“我在逐渐逼近目标岗位”的成就感。
- **2.4 实时信息咨询栏**：滚动展示，利用 AI 抓取并整理与用户目标岗位、兴趣匹配的行业资讯（如电网新闻、最新岗标要求等）。

### 2.3 一站式沉浸知识图谱引擎 (Knowledge Graph Explorer)
**功能目标**：将“五级分层网络”以最直观的方式呈现。摒弃多页面跳转导致的上下文割裂，打造**一站式沉浸工作台 (Master-Detail View)**。
- **3.1 核心图谱主舞台**：全屏展示 2D/3D 拓扑网络（领域 -> 岗位 -> 知识点 -> 场景 -> 步骤）。
- **3.2 全局/个人视角无缝切换 (Toggle)**：在左上角设置切换开关。用户可以在“全局通用图谱”和“我的目标专属图谱”之间一键切换。切换时伴随平滑的节点隐藏/高亮过滤动效，不发生页面跳转。
- **3.3 侧边透视面板 (Sidebar Drawer)**：在图谱中点击任意节点时，优雅地从右侧滑出详情面板（图谱本身向左微缩但保持可见）。展示：
  - AI 给出的学习建议与岗位场景说明。
  - 挂载的资源与前置/后置依赖关系。

### 2.4 设置页
**功能目标**：标准化的用户个性化配置中心与 AI 数据管理。
- **4.1 基础配置**：界面风格/主题颜色选择，深浅模式切换，账号安全（绑定邮箱/第三方账号）。
- **4.2 AI 记忆管理与重置**：展示“AI 目前认为我的水平/偏好”。提供**“我要重新测评”**或“清除特定学习记忆”功能，方便突击学习后的用户刷新初始图谱权重。

### 2.5 沉浸式问答页 (GraphRAG 面板)
**功能目标**：透明展示 AI 推理过程、提供精准溯源的极客级学习辅助工具。
- **5.1 悬浮/画中画模式**：问答窗口不仅支持独立全屏，还支持作为**悬浮组件**在知识图谱页打开，确保边探索图谱边提问的流畅体验。
- **5.2 核心模块组成**：
  - **对话主窗口**：支持 Markdown 丰富渲染。
  - **推理链路展示区**：折叠面板透明展示多智能体流水线（“检索智能体遍历图谱...” -> “推理智能体生成解答...”）。
  - **原文溯源侧栏**：点击引用标记，右侧滑出引用的**知识库原文或企业标准切片**。
  - **追问推荐 (Action Chips)**：结合图谱上下文预测的后续快捷提问标签。

### 2.6 教研反哺页 (教师/教研端)
**功能状态**：规划为**后续开发**。
- 用于提供大纲与企业需求的对比沙盘，以及全班学生群体能力热力图。

## 3. 前端工程选型确认
- **框架**：React + Vite
- **UI 组件库**：shadcn/ui + Tailwind CSS (驱动主题与深浅模式切换)
- **图表与图谱渲染**：
  - 进度/技能树等图表：ECharts
  - 复杂关系网图谱：AntV G6
- **状态管理**：Zustand (管理当前选择的图谱节点、用户全局状态等)
