# 功能模块

## 1. 行业通用与个人双知识图谱
- **行业通用图谱**：构建电力行业专属“技术领域-岗位标准-知识点-应用场景-操作步骤”五级分层网络。涵盖电路基础、电机学、电力系统、新能源智能电网等领域。
- **个人动态图谱**：基于诊断测试与学习行为，为每位用户构建专属图谱，记录各节点掌握度权重，与通用图谱校准，随学习进程实时演化。

## 2. 多智能体引擎 (Multi-Agent System)
- **总控调度智能体**：负责意图识别与任务分发。
- **知识检索智能体**：基于 GraphRAG 执行高效图谱节点穿透与知识提取。
- **推理路径生成智能体**：核心大脑，综合个体图谱与岗位缺口，推理学习路径。
- **知识库更新智能体**：自动化处理新知识并结构化入库。

## 3. 个性化学习路径生成
- **核心逻辑**：以推理智能体为核心，改变“一刀切”推送。
- **输入**：个人图谱掌握度、学习进度、薄弱环节、目标岗位能力要求（产教融合指标）。
- **输出**：生成高度定制化、阶段性的学习计划（包含知识点讲解、应用场景实操方案）。

## 4. 产教融合生态分析
- **资源转化**：将电网、新能源企业真实运维案例与现行标准转化为可教学的节点。
- **教学反哺**：多智能体横向对比岗位能力模型与当前课程大纲体系的缺口，反向输出教研改进建议。

## 5. 专业能力测评与画像
- **动态测评**：题目/任务驱动，结合操作步骤级评估当前水平。
- **画像构建**：输出细粒度的能力层级、偏好、并实时映射至个人图谱。

## 6. 学习资源推荐与深度问答 (GraphRAG)
- **关联推荐**：依据图谱依赖关系，推荐教学视频、案例分析、企业标准文档。
- **关联推理问答**：从“文档检索”跃迁为“知识关联推理”，深挖知识点背后的依赖和场景，生成强可解释、必带权威引用的回答。

## 7. 学习行为追踪与图谱自进化
- **行为追踪**：记录任务完成、资源消耗时长、错误归因。
- **自进化闭环**：数据反馈回知识图谱模块，动态调整个人掌握度，自动触发路径纠偏。
