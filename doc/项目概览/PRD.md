# PRD

## 1. 背景与目标
电气相关专业课程体系复杂、知识依赖强，学生常缺乏清晰的学习路线与阶段判断。本产品以学习路径为主线，结合画像、测评与资源复用型问答，形成可解释的学习闭环。

## 2. 目标用户
- 电气相关专业本科生
- 重点服务上海电力大学学生场景

## 3. 产品目标
- 识别用户学习阶段与知识缺口
- 生成个性化学习计划与路径
- 基于资源库的可追溯专业问答
- 行业资讯与学习内容联动推送

## 4. MVP 范围
### 必做
1. 用户登录与画像采集
2. 知识图谱基础结构
3. 学习路径推荐
4. 专业问答（资源复用）
5. 学习资源推荐

### 暂不做或后期
- 社交与复杂社区
- 付费内容与课程交易
- 多角色复杂审批

## 5. 关键场景
- 新用户完成画像 -> 自动生成学习路径
- 用户完成测评 -> 更新当前阶段与下一步建议
- 用户提问 -> 先检索资源库再生成回答
- 学习进度更新 -> 路径动态调整

## 6. 核心指标
- 画像完成率
- 路径推荐采纳率
- 资源点击与学习完成率
- 问答满意度与引用覆盖率

## 7. 交互原则
- 首页突出学习路径与当前状态
- 问答不是主线
- 图谱是亮点但不喧宾夺主

## 8. 风险与约束
- 不依赖模型自由生成
- AI 输出结构化、可解释
- 业务规则由后端与规则引擎控制
