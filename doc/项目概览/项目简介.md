# 项目简介

## 项目名称
电荔——上电AI学习助手

## 项目定位
面向电气相关专业本科生的个性化学习成长系统。通过用户画像、知识图谱、能力测评、学习路径推荐、行业资讯联动推送和基于资源复用的智能问答，帮助学生明确学习阶段与下一步学习内容。

## 核心价值
- 明确学习阶段与知识缺口
- 生成可解释的个性化学习路径
- 复用资源库进行专业问答
- 资讯与学习内容联动
- 构建学习闭环：测评 -> 推荐 -> 学习 -> 反馈 -> 再推荐

## 核心用户
- 电气相关专业本科生
- 主要面向上海电力大学学生场景

## 范围边界
### 包含
- 用户画像与能力测评
- 学习路径与资源推荐
- 专业问答与行业资讯联动
- 学习进度记录与知识图谱展示

### 不包含或后期
- 社交与复杂社区
- 课程交易/付费内容
- 多角色复杂审批
- 过度复杂多智能体系统
- 泛人群教育平台

## 技术栈概览
- 前端：React + Tailwind CSS + shadcn/ui + Lucide Icons
- 后端：Spring Boot + Spring Security + MySQL + Redis
- AI：Spring AI（ChatClient、Advisor、RAG、Tool Calling）
