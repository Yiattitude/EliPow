# 项目简介

## 项目名称
电荔——上电AI学习助手

## 项目定位
面向电气相关专业本科生的个性化学习成长系统。本平台由电力行业专属知识图谱驱动，以多智能体协同学习引擎为核心，自动完成从学情识别到个性化学习方案输出的完整闭环，帮助学生从传统被动学习转向按个人节奏推进的主动学习模式。

## 核心价值
- **五级分层知识体系与个人图谱**：结合“技术领域-岗位标准-知识点-应用场景-操作步骤”构建通用知识网络，并为每位学习者动态生成专属个人知识图谱。
- **多智能体协同**：通过总控、检索、推理、更新等专职智能体的自主分工与迭代校验，实现从“文档检索”到“知识关联推理”的能力跃迁。
- **基于推理的个性化路径生成**：综合分析掌握度、进度、薄弱环节及目标岗位能力，生成高度定制化的学习路径。
- **产教融合资源生态**：对接企业真实运维案例与现行技术标准，利用多智能体分析岗位与大纲缺口，反向支撑高校优化教学。

## 核心用户
- 电气相关专业本科生
- 主要面向上海电力大学学生场景，辐射电力行业新人

## 范围边界
### 包含
- 用户画像、能力测评与个人知识图谱构建
- 基于 GraphRAG 与多智能体的个性化路径推荐
- 产教融合的行业资源抓取与教学转化
- 资源复用关联问答与知识闭环

### 不包含或后期
- 社交与泛水区社区
- 课程交易/付费内容
- 复杂的人工多角色审批
- 泛人群教育平台

## 技术栈概览
- 前端：React + Tailwind CSS + shadcn/ui + ECharts/AntV G6
- 后端：Spring Boot + Spring Security + MySQL + Neo4j (图数据库) + Redis
- AI：Spring AI + 专职多智能体编排 + Neo4j GraphRAG
