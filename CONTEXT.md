# 业务词汇表 (Glossary)

本文档用于统一项目中的核心业务概念，避免前后端及不同模块间的沟通歧义。**本文档不包含任何实现细节。**

- **行业通用图谱 (Industry General Graph)**：全量知识图谱，包含电力行业所有“技术领域-岗位标准-知识点-应用场景-操作步骤”的五级分层网络数据。
- **个人动态图谱 (Personal Dynamic Graph)**：用户专属的图谱状态。记录每个用户对各个知识节点的掌握度权重，随学习和测评实时演化。
- **岗位能力模型 / 岗位路径 (Role Path)**：行业通用图谱的一个子集视图。它是根据特定目标岗位要求，在全量图谱上圈出的一条学习路径。它**不是**一个独立的图谱。
- **动态学习计划 (Dynamic Study Plan)**：由系统基于“时间预算”、“岗位缺口”和“个人掌握度”自动推断并生成的阶段性（如按周）任务序列，用户仅进行确认和微调，而非从零手动编排。
- **单体苏格拉底 Agent (Single-Agent Socratic MVP)**：V1 版本唯一的核心智能体。它通过一套硬编码的系统提示词（而非复杂多智能体路由）和一系列函数调用（Tool Calling），同时负责知识检索、路径生成建议和苏格拉底式启发问答。
